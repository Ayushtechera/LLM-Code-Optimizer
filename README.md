# LLM Code Optimizer

Convert Python code into optimized C++ using an LLM and compare the execution performance.

<img width="923" height="478" alt="Screenshot 2026-07-16 003803" src="https://github.com/user-attachments/assets/ccc53654-d840-495a-a93e-02b7931ec021" />

## Features

- Converts Python code to C++
- Generates optimized C++ code
- Compiles and runs the generated code
- Compares Python and C++ execution time

## Tech Stack

- Python
- C++
- OpenAI SDK
- Google Gemini
- g++

## Project Structure

```
.
├── project.ipynb
├── .env
├── pyproject.toml
└── uv.lock
```

## Run

1. Install dependencies
2. Add your API key to `.env`
3. Open `project.ipynb`
4. Run all cells

## Output

- Optimized C++ source code
- Compiled executable
- Execution time comparison
